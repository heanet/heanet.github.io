Moodle media plugins


Usage guide

Download the files

  • Repository: heanetmedia_list.zip
  • Repository: heanetmedia_upload.zip
  • Filter: heanetmedia.zip

Install the plugins

  1. go to your plugin installer page (ie: example.ie/admin/tool/installaddon/index.php)
  2. from the drop down, select "Repository (repository)"
  3. choose the file "heanetmedia_list.zip
  4. click the blue button "Install the plugin from the ZIP file"
  5. you should see the next page titled as "Plugin package validation", and verify that it reads "Validation passed!"
  6. click the grey button "Install Plugin!"
  7. you should see the next page titled as "Plugins check"
  8. click the grey button "Upgrade Moodle database now"
  9. you should see the next page title as "Upgrading to new version", and verify that it reads "Success"
  10. click the grey button "Continue"
The steps are analogous for the other two plugins

Upload a file

  1. in any editor with the media button, click the button
  2. a window named "Insert media" will open
  3. click the grey button "Browse repositories"
  4. on the left, among other repositories you might have, you will see two prefixed "Heanet Media:"
  5. click the repostiory "Heanet Media: Upload"
  6. you will be shown the IDP Authentication Page
  7. you sill see the media.heanet.ie upload page
  8. choose a file to upload and click the blue button "Upload File"

List Uploaded files

  1. in any editor with the media button, click the button
  2. a window named "Insert media" will open
  3. click the grey button "Browse repositories"
  4. on the left, among other repositories you might have, you will see two prefixed "Heanet Media:"
  5. click the repostiory "Heanet Media: List"
  6. the first time you click it, media will attempt to link your moodle account to your media user id
  7. either the list of your videos will appear, or a confirmation page will
  8. if you see the confirmation page, simply click "Heanet Media: List" on your left side, to try again.
