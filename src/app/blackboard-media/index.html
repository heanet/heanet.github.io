Blackboard Media Building Block


Usage guide

Download the plugin file

  • Repository: blackboard_media

Install the plugin

  1. From the Administration Panel, go to Building Blocks -> Building Blocks -> Installed Tools -> Upload Building Blocks.
  2. You can then select the heanet_media.war file as the package to upload and continue through the installation (you should have received it by e-mail). The way to update the plugin will be the same. This should give you a successful install message.
  3. As it is possible that your Blackboard instance doesnt enable fully the plugin, I would invite you to ensure that the Course tool is active (from Admin Panel Tools and Utilities Tools).
  4. (Optional) It isnt really required to test with an Admin account but if later on you want to allow everyone to add video content from media, you may need to follow this process to add media.heanet.ie as a trusted source (ortherwise Blackboard will filter the embed media player for some roles). The line to add in the policy file would be: <regexp value="http[s]?://media.heanet.ie/.*" />

Upload a file

  1. In any editor you may find within a Community forum or Courses, choose Mashups -> Heanetmedia Upload
  2. a pop-up window to Heanet Media will open
  3. first with the IDP Authentication Page
  4. then with media.heanet.ie upload page
  5. choose a file to upload and click the blue button "Upload File"


Mashup Selection

List Uploaded files

  1. In any editor you may find within a Community forum or Courses, choose Mashups -> Heanetmedia List
  2. The first time you click it, a pop-up will open and media will attempt to link your Blackboard account to your media user id
  3. either the list of your videos will appear, or a confirmation page will
  4. If you see the confirmation page, simply reload the pop-up to try again
  5. Once you have the list of videos, simply select one (a blue contour will appear) and click on "Add Video"
  6. The window will close and you will be able to see the media player on your content
  7. Save your content, you can then watch the selected video with the media player
